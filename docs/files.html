integer.js

Defines an exports.Integer class for representing (potentially) infinite length two's-complement integer values. For the specific case of 64-bit integers, use Long, which is more efficient.

long.js

Defines a exports.Long class for representing a 64-bit two's-complement integer value, which faithfully simulates the behavior of a Java "long". This implementation is derived from exports.LongLib in GWT.
